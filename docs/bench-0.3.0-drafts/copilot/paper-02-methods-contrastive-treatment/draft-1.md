## Methods

We model each input as an attributed graph \(G=(V,E,X)\), where \(X \in \mathbb{R}^{|V|\times d}\) denotes node features. The graph encoder maps \(G\) to a graph-level embedding \(z \in \mathbb{R}^{128}\) through three stacked graph isomorphism network layers with hidden size 128, each followed by batch normalization, ReLU activation, and 0.2 dropout. Node representations from the final layer are aggregated with sum pooling and passed to a two-layer projection head with output dimension 64 for contrastive training. We use Adam with a learning rate of \(1\times 10^{-3}\), weight decay \(1\times 10^{-5}\), and batch size 64. This architecture was chosen because sum pooling preserves graph-level count information while the projection head separates the contrastive space from the encoder space used at inference.

Pretraining is fully unsupervised and optimizes a graph-level contrastive objective. For each graph, we generate two correlated views by applying stochastic augmentations that preserve coarse semantic structure while perturbing local evidence: node dropping with probability 0.1, edge perturbation at rate 0.1, and feature masking on 15% of node attributes. The two augmented views of the same base graph form a positive pair. Negative pairs are constructed implicitly from the other graphs in the same mini-batch, so each anchor has \(2(B-1)\) negatives when the batch contains \(B\) original graphs. Let \(h_i\) and \(h_i^+\) denote the projected embeddings of a positive pair. We maximize agreement with the normalized temperature-scaled cross-entropy loss, using cosine similarity and temperature \(\tau=0.2\). To reduce representation collapse, all embeddings are \(L_2\)-normalized before similarity computation. Pretraining runs for 200 epochs with early stopping based on the moving average of the contrastive loss over the validation split.

At inference, anomaly detection is performed without labels by measuring how far a test graph lies from the distribution of normal training embeddings. We discard the projection head, compute encoder embeddings for all training graphs, and fit a Gaussian reference model in the 128-dimensional encoder space using the empirical mean and covariance with a shrinkage coefficient of 0.01 for stability. For a test graph embedding \(z\), the anomaly score is its Mahalanobis distance to this reference distribution. Higher scores indicate lower conformity to the learned normal manifold. We rank graphs by this score and use the raw distance as the final anomaly measure.
