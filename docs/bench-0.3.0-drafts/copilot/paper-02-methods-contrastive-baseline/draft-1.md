## Methods

We model each input as a graph \(G=(V,E,X)\), where \(X\) denotes node attributes. The graph encoder is a three-layer Graph Isomorphism Network (GIN) with hidden dimension 128 and ReLU activations, followed by batch normalization and a mean-pooling readout to obtain a graph-level embedding \(h_G \in \mathbb{R}^{128}\). Each GIN layer updates node states by aggregating neighbor messages and applying a multilayer perceptron, which provides sufficient expressive power to capture higher-order structural irregularities while remaining stable on small and medium-sized graphs. A two-layer projection head then maps \(h_G\) to a 64-dimensional contrastive space used only during pretraining. We train the encoder with Adam using a learning rate of \(10^{-3}\), weight decay of \(10^{-5}\), dropout of 0.2, and batch size 64.

The encoder is first optimized by contrastive self-supervised pretraining on unlabeled graphs. For each graph, we generate two positive views by applying stochastic augmentations that preserve semantic identity while perturbing superficial structure: node feature masking with rate 0.1, edge dropping with rate 0.15, and random subgraph sampling that retains 85% of nodes. The two augmented views of the same original graph form a positive pair. Negative samples are all other graphs in the minibatch, following the in-batch contrastive setting; thus, each anchor is contrasted against \(2(B-1)\) negatives when the batch size is \(B\). We optimize the normalized temperature-scaled cross-entropy (NT-Xent) loss on cosine similarities between projected embeddings, with temperature \(\tau=0.2\). Pretraining runs for 200 epochs, and the model with the lowest validation loss is retained. This objective encourages invariance to benign graph perturbations while separating representations of distinct graph instances, which is desirable for anomaly-sensitive representation learning without labels.

At inference, the projection head is discarded and the pooled encoder embedding \(h_G\) is used to compute an anomaly score. We estimate a reference center \(c\) as the mean embedding over the training graphs, and define the anomaly score for graph \(G\) as \(s(G)=1-\cos(h_G,c)+\lambda\|h_G-c\|_2^2\), with \(\lambda=0.5\). The cosine term captures angular deviation from the normality prototype, while the Euclidean term penalizes large radial departures. Higher scores indicate greater abnormality, and scores are min-max normalized within each evaluation split for comparability across datasets.
