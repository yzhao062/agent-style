This pull request removes the legacy cookie-based authentication middleware to consolidate identity verification into a single workflow. All API endpoints now require header-based JSON Web Tokens (JWT) as the exclusive authorization scheme. Callers must transition to the `Authorization: Bearer <token>` format before the final migration deadline on June 30, 2026. Clients attempting to use legacy cookies will receive a `401 Unauthorized` response containing a deprecation notice and a link to the migration guide. Existing service logs will record these failures as deprecated auth attempts to help teams track remaining legacy dependencies.
