## Methods
The proposed framework utilizes a multi-layer Graph Isomorphism Network (GIN) as the primary encoder architecture to transform input graphs into a low-dimensional latent space. The encoder is composed of four graph convolutional layers, each featuring a hidden dimensionality of 128 and utilizing jumping knowledge connections to capture both local and global structural nuances. Each GIN layer implements a permutation-invariant aggregation function followed by a multi-layer perceptron with ReLU activations and batch normalization. For an input graph $G$, the final graph-level representation is obtained through a global sum pooling operation. During initialization, we apply a learning rate of $10^{-3}$ and a weight decay of $10^{-5}$ to the Adam optimizer to ensure the learned embeddings are sufficiently regularized and stable across training iterations.

The core of our approach is a contrastive self-supervised pretraining objective that encourages the model to learn invariant graph features without manual labels. For each graph in a training batch of size 256, we generate two augmented views by applying stochastic transformations: node dropping and edge perturbation. Node dropping randomly removes 20% of the vertices, while edge perturbation modifies the connectivity by adding or deleting 10% of the edges. These views are processed by the shared encoder to generate embedding pairs, which are evaluated using the normalized temperature-scaled cross-entropy (NT-Xent) loss. We set the temperature hyperparameter $\tau$ to 0.07 to control the sharpness of the similarity distribution. By maximizing the agreement between positive pairs and minimizing it against negative samples, the encoder learns to map normal graphs to a cohesive region of the representation space.

During the inference stage, anomaly detection is executed by measuring the deviation of a test graph's embedding from the normative distribution established during pretraining. We define the anomaly score as the Euclidean distance between the test graph's latent vector and the centroid of the training embeddings. This centroid is computed as the mean of all normal graph representations. A test graph is classified as anomalous if its score exceeds a threshold derived from the 95th percentile of the training score distribution. This scoring mechanism effectively identifies anomalies by exploiting the fact that graphs with irregular structural patterns fail to align with the compact representations of normal data, resulting in significantly higher distance metrics in the latent space.
