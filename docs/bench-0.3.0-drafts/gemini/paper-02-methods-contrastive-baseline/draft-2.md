## Methods

Our framework utilizes a Graph Isomorphism Network (GIN) as the backbone encoder to extract expressive graph-level representations. The encoder consists of five sequential GIN layers, each followed by Batch Normalization and ReLU activation, allowing the model to capture complex structural dependencies and distinguish between various graph isomorphisms. We employ a global sum-pooling operator to aggregate node-level features into a fixed-length graph embedding vector, which effectively preserves the structural properties of the input graphs regardless of their size. To map these embeddings into a latent space suitable for contrastive learning, a two-layer Multi-Layer Perceptron projection head is applied. During the training phase, the model is optimized using the Adam optimizer with a constant learning rate of $1 \times 10^{-3}$, ensuring stable convergence across diverse and potentially noisy graph topologies.

The unsupervised pretraining phase is governed by a contrastive objective that encourages the encoder to learn invariant features through self-supervision without the need for manual labels. For each graph in a mini-batch, we generate two correlated views, or positive samples, by applying stochastic augmentations: random node dropping, which removes 15% of the nodes and their incident edges, and attribute masking, which hides a portion of the node feature vectors. Negative samples are defined as the representations of all other graphs within the same mini-batch. We minimize the Normalized Temperature-scaled Cross Entropy (NT-Xent) loss, which maximizes the cosine similarity between positive pairs while pushing negative samples apart in the latent space. This objective is sensitive to the temperature hyperparameter $\tau$, which we set to 0.15 to control the sharpness of the similarity distribution, and we utilize a batch size of 256 to provide a sufficiently diverse set of negative examples for each anchor graph.

At inference, the anomaly score for a test graph is determined by its local density relative to the distribution of normal graphs in the learned embedding space. We first compute the embeddings for all training graphs using the pretrained encoder and store them as a reference representation of the normal class. For a given test graph, we project it into the latent space and calculate its anomaly score based on the average Euclidean distance to its $k$-nearest neighbors within the training set. Graphs that exhibit structural or feature-based irregularities will naturally reside in low-density or peripheral regions of the latent space, resulting in significantly higher distance values compared to typical samples. For our experiments, we set the neighborhood size $k=10$ to ensure the score is robust against individual outliers in the training distribution while maintaining high sensitivity to global structural anomalies that deviate from the learned manifold.
