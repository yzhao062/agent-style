## Methods

We employ a Graph Isomorphism Network (GIN) to extract high-dimensional representations from input graphs of varying sizes. This architecture consists of five message-passing layers that use a sum-aggregation mechanism to maintain injective mapping properties. Each layer transforms node features through a two-layer MLP with batch normalization and ReLU activation functions to capture non-linear local patterns. After the final convolution, a global sum-pooling operation aggregates node embeddings into a fixed-length graph-level vector. Layer-wise dropout of 0.1 prevents overfitting on the graph topology during the training phase. Xavier initialization and a learning rate of 0.001 for the Adam optimizer ensure stable convergence of the network parameters.

The training process utilizes a contrastive self-supervised objective to learn invariant features from unlabeled data. We generate two correlated views of each graph through stochastic augmentations, specifically node dropping and edge perturbation with a 0.2 probability. Augmented versions of the same input form a positive pair, while all other graphs within the same mini-batch serve as negative samples. We project the graph-level vectors into a lower-dimensional latent space using a non-linear projection head before computing the loss. Optimization relies on the InfoNCE objective to maximize the agreement between positive pairs and increase the distance between negative samples. This calculation incorporates a temperature parameter $\tau = 0.1$ to adjust the sensitivity of the softmax distribution. We set the training batch size to 128 to provide a sufficient number of negative samples for robust representation learning.

At inference time, the framework identifies anomalies by measuring the deviation of a test graph from the learned distribution of normal instances. We compute an anomaly score based on the cosine similarity between the embeddings of the original graph and multiple augmented variations. High-quality representations of normal graphs exhibit high consistency across different views, whereas anomalous structures produce divergent embeddings. Intermediate representations pass through a hidden layer with a dimension of 512 to stabilize the scoring metric. A lower similarity score indicates a higher probability that the input graph represents an out-of-distribution sample. The final detection process uses a threshold-based approach on these normalized scores to flag potential irregularities. This scoring mechanism allows the system to detect structural and feature-based anomalies without requiring explicit labels during the pretraining phase.
