## Methods
The proposed framework employs a Graph Isomorphism Network (GIN) to extract permutation-invariant graph-level representations from input data. This encoder maps an input graph $\mathcal{G} = (\mathcal{V}, \mathcal{E})$ to a fixed-dimensional vector $\mathbf{h}_{\mathcal{G}}$ through five successive message-passing layers. Each layer aggregates neighbor features using a multi-layer perceptron (MLP) with 64 hidden units, ensuring the model distinguishes diverse local structures and maintains injective properties. We initialize the model parameters with a learning rate of 0.001 and optimize the weights using the Adam algorithm. Global sum pooling aggregates node-level embeddings into a unified graph representation, providing a compact descriptor that captures the global topology of the input graph.

The training phase utilizes a self-supervised contrastive objective to learn robust features from the data distribution without manual labels. We generate two correlated views of each graph through stochastic augmentations, including node dropping and edge perturbation with a modification ratio of 0.15. Positive pairs consist of these augmented versions derived from the same original graph, while negative samples comprise all other graphs within the current training batch. The model passes the pooled embeddings through a non-linear projection head before minimizing a normalized temperature-scaled cross-entropy loss. We set the temperature hyperparameter $\tau$ to 0.07 to sharpen the similarity distribution and force the model to learn more discriminative features. This process aligns the representations of similar graphs while pushing apart disparate structural patterns in the latent space.

During inference, the system detects anomalies by measuring the deviation of a query graph from the manifold established during pretraining. We calculate an anomaly score based on the Euclidean distance between the query embedding and the centroid of the training embeddings in the feature space. Graphs that reside far from the dense clusters of normal training samples receive higher scores, indicating a low probability of belonging to the learned distribution. We execute this inference process using a batch size of 256 to stabilize the distribution of embeddings and ensure efficient computation. The resulting anomaly scores provide a continuous ranking, where the highest values pinpoint graphs with irregular connectivity or feature distributions that suggest potential system failures or malicious activity.
