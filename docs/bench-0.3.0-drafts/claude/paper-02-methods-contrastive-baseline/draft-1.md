## Methods

Our approach consists of three components: a graph encoder that maps entire graphs to a shared latent space, a contrastive pretraining objective that shapes this space without labels, and a density-based anomaly score computed at inference.

The encoder is a five-layer Graph Isomorphism Network (GIN) with hidden dimension 128 and ReLU activations, followed by a sum-pooling readout and a two-layer MLP projection head that maps pooled features to a 64-dimensional embedding that is L2-normalized. Each GIN layer applies the standard update $h_v^{(k)} = \text{MLP}^{(k)}\big((1+\epsilon^{(k)}) h_v^{(k-1)} + \sum_{u \in \mathcal{N}(v)} h_u^{(k-1)}\big)$, with $\epsilon^{(k)}$ learned jointly with the MLP weights. Batch normalization is applied between layers and dropout of 0.2 is used within the projection head. Input node features are concatenated one-hot degree encodings and any dataset-provided categorical attributes.

For pretraining we adopt a GraphCL-style objective built on stochastic augmentations. For each graph $G$ in a minibatch of size 256, two views $\tilde{G}_1, \tilde{G}_2$ are generated by independently composing node dropping (ratio 0.1), edge perturbation (ratio 0.1), and attribute masking (ratio 0.2). The two views of the same graph form the positive pair, while the remaining $2(B-1)$ projected views in the batch serve as in-batch negatives. We minimize the normalized temperature-scaled InfoNCE loss
$$\mathcal{L} = -\log \frac{\exp(\text{sim}(z_i, z_j)/\tau)}{\sum_{k \neq i}\exp(\text{sim}(z_i, z_k)/\tau)},$$
where $\text{sim}$ denotes cosine similarity and the temperature is fixed at $\tau = 0.5$. Training runs for 100 epochs with Adam at learning rate $1\times 10^{-3}$, weight decay $1\times 10^{-5}$, and cosine annealing.

At inference, encoder weights are frozen and all graphs from the clean training split are embedded to form a reference set. The anomaly score for a test graph is its mean cosine distance to its $k=10$ nearest neighbors in the reference set, $s(G) = \tfrac{1}{k}\sum_{z' \in \mathcal{N}_k(z)} (1 - z^\top z')$. Graphs that fall in sparsely populated regions of the learned manifold receive high scores; the detection threshold is calibrated on a held-out validation split to match a target false-positive rate of 5%.
