## Experiments

We evaluated the proposed detector on five benchmark graph anomaly datasets. Cora is a citation network in which papers form nodes, citation links form edges, and topic labels allow rare-class nodes to serve as contextual anomalies. Citeseer is another citation graph, but it is sparser than Cora and tests whether a detector can separate anomalies when homophily is weaker. Pubmed is a larger biomedical citation network that stresses scalability because it contains more nodes and longer-range neighborhood patterns. BlogCatalog is a social network of bloggers, where group memberships provide semantic context and anomalous users often connect across unrelated communities. Flickr is a user-photo social graph whose dense local communities make structural anomalies difficult to distinguish from high-degree normal users.

For each dataset, we used a fixed 60/20/20 node split for training, validation, and testing, stratified by anomaly label where labels were available. The model used only graph structure and node attributes during training; anomaly labels were reserved for validation-based early stopping and final testing. We selected hyperparameters on the validation split and reported test performance once per run. ROC-AUC was the primary metric because it measures ranking quality without requiring a deployment threshold. We also tracked PR-AUC because the anomaly rate ranged from 3.1% on Pubmed to 12.4% on BlogCatalog. To estimate variance, we repeated every experiment with ten random seeds, changing weight initialization, negative sampling, and split generation, and report mean scores with standard deviations.

The proposed model achieved the best average ROC-AUC, 91.8 ± 0.6, and the best average PR-AUC, 64.2 ± 1.4. It outperformed DOMINANT by 5.1 ROC-AUC points on average, with the largest gain on Citeseer, 90.6 versus 83.9, where sparse structure made reconstruction error less reliable. It also improved over CoLA by 3.4 points and ANEMONE by 2.7 points, suggesting that combining attribute reconstruction with neighborhood contrast captures both contextual and structural anomalies. On the larger Pubmed graph, the method reached 93.1 ROC-AUC, compared with 90.4 for GRADATE and 88.2 for CoLA, while using 27% less training time than GRADATE. The strongest ablation removed the structural-context contrastive branch; average ROC-AUC fell from 91.8 to 88.2, and PR-AUC fell from 64.2 to 58.7. This drop shows that the contrastive branch provides the main signal for identifying nodes whose local wiring conflicts with their attributes.
