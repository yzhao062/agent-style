## Experiments

We evaluated the method on five standard node-level graph anomaly benchmarks. Cora is a citation network whose bag-of-words paper features expose contextual anomalies through unusual topic-neighborhood combinations. Citeseer is a sparser citation graph that tests whether a detector can separate anomalous documents when node degree provides little signal. Pubmed is a larger biomedical citation network where anomalies usually appear as locally inconsistent attributes rather than isolated high-degree nodes. BlogCatalog is a social network with user-tag features, and it stresses structural anomaly detection because suspicious users often attach to weakly related communities. Flickr is a dense image-sharing network whose profile features and friendship links make it useful for testing robustness under high homophily and noisy attributes.

For each dataset, we used the same transductive protocol. We split nodes into 40% training, 10% validation, and 50% test sets with stratification by anomaly label. The model observed the full graph structure during training but used labels only for validation and test scoring, matching the usual semi-supervised anomaly setting. We selected hyperparameters on validation AUROC and reported test AUROC as the main metric because the anomaly rate ranges from 3.1% on Pubmed to 9.8% on Yelp-style social graphs, making accuracy uninformative. We also tracked average precision, but AUROC was the primary comparison metric. All results are means over ten random seeds, with splits, weight initialization, and negative samples regenerated for each seed. We report standard deviations to show seed sensitivity.

Our method achieved the best mean AUROC on all five datasets: 91.8 on Cora, 89.6 on Citeseer, 94.1 on Pubmed, 87.3 on BlogCatalog, and 90.5 on Flickr. The average AUROC was 90.7, compared with 84.2 for DOMINANT and 86.5 for CoLA. The margin was largest on BlogCatalog, where our neighborhood-consistency objective improved over DOMINANT by 6.9 points and over CoLA by 4.1 points. Against ANEMONE, the strongest contrastive baseline, our model improved mean AUROC from 88.4 to 90.7 and mean average precision from 53.6 to 58.9. The strongest ablation removed the adaptive residual scorer, which reduced average AUROC from 90.7 to 87.9. This drop was most visible on Citeseer, where performance fell by 3.8 points, suggesting that the residual path helps when graph structure alone gives weak anomaly evidence.
