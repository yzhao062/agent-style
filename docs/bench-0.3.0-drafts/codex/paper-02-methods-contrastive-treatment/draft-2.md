## Methods

We encode each input graph with a three-layer graph isomorphism network that maps node attributes and topology to a fixed-length graph embedding. Each layer applies a sum aggregator followed by a two-layer multilayer perceptron with batch normalization and ReLU activation. The hidden width is 128 in all message-passing layers, and dropout with probability 0.2 follows each activation. Edge attributes, when present, enter through a learned linear projection added to the source-node message. After message passing, we obtain graph representations by concatenating global sum pooling and global mean pooling, then project the 256-dimensional pooled vector through a 128-dimensional projection head during pretraining.

We train the encoder with a contrastive self-supervised objective that treats two stochastic views of the same graph as a positive pair. For each graph, we sample two augmentations by independently dropping 15% of edges, masking 10% of node features, and extracting an induced subgraph that retains 80% of nodes through random-walk sampling. These perturbations preserve coarse graph structure while forcing the encoder to ignore small local changes. All other augmented graphs in the same mini-batch serve as negatives. Let \(z_i\) and \(z_j\) denote L2-normalized projection-head outputs for two views of the same graph. We minimize the normalized temperature-scaled cross-entropy loss, with cosine similarity in the numerator for the positive pair and cosine similarities to all negatives in the denominator. The temperature is 0.2, the batch size is 128 graphs, and optimization uses Adam with learning rate \(10^{-3}\), weight decay \(10^{-5}\), and 200 pretraining epochs.

At inference, we discard the projection head and use the encoder output before projection as the graph embedding. Since no anomaly labels are used, we define normality through the density of training embeddings. For a test graph \(G\), we compute its normalized embedding \(h_G\) and retrieve its \(k=20\) nearest training embeddings under cosine distance. The anomaly score is the mean distance to these neighbors, rescaled by the median and interquartile range of training-set neighbor distances. Larger scores indicate graphs that lie in sparse regions of the learned representation space. We report anomaly rankings directly from this score and select thresholds only for evaluation metrics that require binary decisions.
