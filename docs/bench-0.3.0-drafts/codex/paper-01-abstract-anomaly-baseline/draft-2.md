Graph-level anomaly detection seeks to identify rare, structurally or semantically atypical graphs without labeled outliers, a setting that arises in molecular screening, transaction monitoring, and network security but remains challenging because normal graphs can be heterogeneous and anomalies may differ only through subtle substructures. We propose Graph Contrastive Anomaly Pretraining (GCAP), an unsupervised framework that first learns graph representations by contrasting multiple topology- and attribute-preserving views of each graph against hard negatives drawn from the same minibatch, then fits a compact one-class density model in the learned embedding space. To avoid teaching invariance to anomalous evidence, GCAP uses adaptive augmentations that preserve high-mutual-information motifs while perturbing low-confidence edges, and it weights contrastive pairs by an estimated normality score updated during training. Across six TU-style graph anomaly benchmarks, GCAP improves mean AUROC over the strongest unsupervised baseline from 82.7 to 88.5, including gains of 6.3 points on MUTAG and 5.1 points on PROTEINS. On an OGBG-moltox21 rare-toxicity detection protocol, GCAP achieves 79.4 AUROC, outperforming GraphCL followed by Deep SVDD by 4.2 points while using no anomaly labels. Under 1% synthetic contamination on IMDB-BINARY and COLLAB, GCAP reduces false-positive rate at 90% recall by 12.6 and 9.4 points, respectively. These results suggest that contrastive pretraining, when coupled with anomaly-aware view construction, provides a robust foundation for unsupervised graph-level outlier detection.
