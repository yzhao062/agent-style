## Methods

We use a graph encoder that maps each input graph to a fixed-dimensional representation through stacked message-passing layers followed by permutation-invariant pooling. Each node is initialized with its attributes; when attributes are unavailable, we use normalized degree and a constant indicator as structural features. The encoder is a 3-layer Graph Isomorphism Network with 128 hidden units per layer, batch normalization, ReLU activations, and dropout of 0.2 after each layer. Node embeddings from all layers are concatenated to preserve both local and higher-order structural information, then aggregated using mean and max pooling. The resulting graph vector is passed through a two-layer projection head with output dimension 64 during pretraining; for downstream anomaly scoring, we use the representation before the projection head. The encoder is optimized with Adam using a learning rate of 1e-3 and weight decay of 1e-5.

Contrastive self-supervised pretraining is performed by constructing two correlated views of each graph and training their embeddings to agree while separating embeddings of different graphs. For each input graph, a positive pair is generated using independent stochastic augmentations: edge dropping with probability 0.15, node-feature masking with probability 0.20, and random-walk subgraph sampling retaining 80% of nodes when the graph has more than 30 nodes. These transformations are chosen to preserve graph-level semantics while perturbing superficial structure and attributes. Negative samples are all other augmented graphs in the same minibatch, under the assumption that distinct graphs should not collapse to identical representations. We minimize the normalized temperature-scaled cross-entropy loss over both views, using cosine similarity between projected embeddings and a temperature of 0.2. Training uses a batch size of 256 graph views, corresponding to 128 original graphs, for 100 epochs with early stopping based on stabilization of the contrastive loss.

At inference, each graph is encoded once without stochastic augmentation, and anomaly scores are computed in the learned representation space. Because anomalies are assumed to be rare, the training embeddings define an empirical reference set for normal graph structure. For a test graph with embedding z, we compute the average Euclidean distance to its k nearest neighbors in the training set, using k = 10 after L2 normalization of embeddings. The anomaly score is therefore s(G) = k^{-1} sum_{i in N_k(z)} ||z - z_i||_2. Larger scores indicate graphs that lie in sparse regions of the learned manifold and are more likely to be anomalous.
