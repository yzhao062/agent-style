## Methods

Each graph is encoded with a message-passing neural network that maps variable-size graphs to fixed-dimensional representations. We use a 3-layer Graph Isomorphism Network encoder with sum aggregation, batch normalization after each message-passing layer, ReLU activations, and dropout with rate 0.2. Node attributes are first projected to a 128-dimensional hidden space; when attributes are unavailable, we use degree embeddings capped at degree 50. The graph-level representation is obtained by concatenating mean and max pooled node embeddings from the final layer, followed by a two-layer projection head with hidden size 256 and output dimension 128. The encoder is trained with Adam using weight decay of 1e-5, and gradients are clipped to norm 5.0 to stabilize training on large graphs.

Contrastive self-supervised pretraining is performed without anomaly labels. For each graph in a minibatch, we generate two correlated views by independently applying stochastic graph augmentations: node-feature masking with probability 0.15, edge dropping with probability 0.10, and subgraph sampling that retains 80% of nodes while preserving the induced edge set. The two views derived from the same original graph form a positive pair. Views derived from different graphs in the same minibatch are treated as negatives. Let z_i and z_j denote L2-normalized projected embeddings of a positive pair. We optimize the normalized temperature-scaled cross-entropy loss, maximizing their cosine similarity relative to all in-batch negatives. The temperature is fixed to 0.2, the batch size is 128 graphs, and pretraining is run for 200 epochs with a learning rate of 1e-3 using cosine decay after a 10-epoch warmup. The projection head is used only during pretraining; downstream anomaly detection uses the encoder representation before projection.

At inference, the pretrained encoder is frozen and each training graph is embedded once to define the nominal reference distribution. For a test graph, we compute its embedding and assign an anomaly score based on distance from this reference set. Specifically, we use the average Euclidean distance to its k nearest training embeddings, with k = 10, after normalizing all embeddings to unit length. To improve robustness to augmentation noise, the final test embedding is averaged over M = 8 independently augmented views using the same augmentation probabilities as pretraining. Larger scores indicate lower agreement with the learned normal graph manifold and are therefore interpreted as more anomalous.
