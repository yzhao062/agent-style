Aider reads a project conventions file when explicitly attached:

```bash
aider --read CONVENTIONS.md
```

Or via `.aider.conf.yml`:

```yaml
read:
  - CONVENTIONS.md
```

Aider is also listed on `agents.md/` as an AGENTS.md-aware tool; the root `AGENTS.md` is picked up automatically. Use either path depending on your project setup.
